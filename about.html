About Sentinel Node Tester

Sentinel is a decentralized privacy network made of independent nodes that carry encrypted traffic. This tool measures every node continuously so you know, at a glance, which nodes are healthy, fast, and reachable right now.

What this tester does

The tester scans the full active node list on the Sentinel chain, then connects to each node, sends real traffic through it, and records the result. This is not a chain-only lookup — it is an end-to-end test of the handshake, the tunnel, and the throughput.

A full scan of the network is called a batch. Batches run back-to-back during public testing so the numbers shown are always based on recent, live evidence.

What the numbers mean

Protocol
The VPN transport the node advertises — WireGuard or V2Ray.
Connection test
Did the cryptographic handshake with the node complete successfully?
Speed
Measured download throughput over the live tunnel, in Mbps.
Recent uptime
How often recent tests have succeeded for this node.
Speed over time
A sparkline showing how throughput has moved across recent batches.
Peers
Active connected clients reported by the node at test time.

Where the data comes from

The node list is pulled from the Sentinel chain (v3 LCD /sentinel/node/v3/nodes, with RPC-first fallback across multiple public endpoints). Every result on this site is generated by a real session on that node — nothing is synthetic, and nothing is cached from third parties.

What is never shown to the public

The admin wallet, plan IDs, fee-grant internals, and any operator-side controls stay on the admin surface. This public site is a read-only view of network health. There is nothing to click, nothing to start, and nothing to pay — by design.
